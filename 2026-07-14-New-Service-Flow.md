# 260714 - 새로운 설계 서비스 플로우

## 설명
새로운 재설계는 실체적인 정량적 데이터가 많아야 하는 프로모션 검토가 아닌, 운영 중인 매장 정보 / 재고 최적화 원하는 품목 / 기존 업계와 사내표준으로 정해져 있던 Lead Time과 Re-Order Point를 매장 특화 정보를 기반으로 ROP 재조정값을 추천하고, 재조정의 이유를 사전에 Agent AI를 통해 축적된 지식 베이스와 사전에 파라미터값으로 설정된 매장 정보와 특성에 가중치를 두고 내부적으로 연산하여 가장 적합한 값을 찾는 것이 핵심.
