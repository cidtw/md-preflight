매장 맞춤 발주 기준

    우리 매장에 맞는 발주 기준을
    다시 잡아 드립니다

    공급 리드타임은 계약·표준값 그대로 둡니다.

    업계·사내 평균만으로는 놓치기 쉬운 부분을
    매장 유형·규모·상권·접근성으로 반영해
    재발주점(ROP)·여유 재고·1회 발주량·발주 주기를 제안하고,
    그렇게 나온 근거까지 함께 보여 줍니다.

    • 입력은 세 단계입니다. 기본 정보 → 세부 정보 → 품목.
    • 결과는 쉬운 설명으로 먼저 보여 주고,
      필요할 때만 전문 해설로 바꿀 수 있습니다.
    • 매장 유형과 규모·객단가가 맞지 않으면,
      규모·객단가 선택을 기준으로 안내합니다.

    시연 · 세솔로 25 주변 실매장

    기준 주소는 경기도 고양시 덕양구 세솔로 25입니다.
    이 주소 주변에서 모은 실매장으로 바로 분석을 돌려 볼 수 있습니다.

    조사 범위 · 편의점 1km · 슈퍼·SSM 3km · 대형마트 10km

    입지·상권·유동은 주변 시설 정보로 추정한 값입니다.
    POS 판매 실측은 아닙니다.

    Kakao Local API FastAPI · Python Vercel 주변 시설 기반 입지 추정

    탐색 · 더미 / 지도 검색

    행정동·주소 입력을 시험해 보는 탐색용 프리셋입니다.

    지도 점포 검색은 위저드에서 「정확한 위치」를 켠 뒤에도 사용할 수 있습니다. 시연 본편은 아닙니다.

    세션 1 / 3

    매장 기본 정보

    채널 체급을 정하는 핵심 값입니다. 유형·규모·객단가를 선택해 주세요.

    세션 2 / 3

    매장 세부 정보

    입지·상권·접근성은 물류 리스크 버퍼와 수요 변동 반영에 쓰입니다. (LT는 고정)

    세션 3 / 3

    품목 · 운영 기준

    품목·소진량과 함께 서비스 레벨·발주 요일을 선택하세요. 품목별 LT는 입력으로 유지되며 결과에서 LT 변동을 추천하지 않습니다. 표준 ROP는 비워 두면 채널 기본값으로 계산합니다.

    매장 특화 점수를 연산하고 지식 베이스를 매칭하는 중…
